<Section icon={Target} title="Rerolls de Status e Sistema de Potencial">

O núcleo do sistema de **status de Expedições de Anime** reside no menu de Potencial de Status. Quando você seleciona uma unidade na seção de melhoria, verá uma lista de suas classificações de potencial atuais. Essas classificações não são permanentes; você pode usar Rerolls de Status para apostar em porcentagens melhores. Isso é particularmente importante para unidades Míticas que você planeja usar em Raides ou Expedições de alto nível.

| Tipo de Item | Uso Principal | Método de Obtenção |
| :--- | :--- | :--- |
| **Reroll de Status** | Altera todos os status desbloqueados em uma unidade | Loja de Ouro, Códigos, Recompensas de Raide |
| **Bloqueio de Status** | Impede que um status específico seja alterado | Loja de Ouro, Eventos Especiais |
| **Reroll de Traço** | Altera o Encantamento/Traço da unidade | Criação, Loja de Música, Raides |
| **Transferência de Shiny** | Move o status shiny para uma unidade normal | Seção de Melhoria (Transferência de Shiny) |

Ao dar reroll, o jogo gera novos valores para cada slot que não esteja bloqueado. Se você conseguir um status de velocidade de ataque de nível "S", deve usar imediatamente um Bloqueio de Status antes de rolar novamente. Tenha em mente que o uso de bloqueios aumenta o custo dos rerolls subsequentes, portanto, priorize bloquear apenas os status mais impactantes para aquele tipo específico de unidade.

<Callout type="warning" title="Gerenciamento de Recursos">
Bloqueios de Status são mais raros que Rerolls. Use-os apenas em unidades Míticas ou Secretas. Usá-los em unidades Raras ou Épicas é geralmente considerado um desperdício de recursos no início do jogo.
</Callout>

</Section>

<Section icon={Swords} title="Mecânicas de Transferência de Status">

Um dos recursos mais amigáveis ao jogador em relação aos **status de Expedições de Anime** é a capacidade de transferir status entre unidades. Se você gastou dezenas de rerolls para deixar uma unidade lendária com status perfeitos, mas depois invocou uma versão Mítica mais poderosa do mesmo personagem, você não precisa recomeçar do zero. O sistema de Transferência de Status permite migrar esses atributos de alto nível diretamente.

<Steps>
  <Step num="1" title="Acesse o Menu de Melhoria">
    Navegue até a seção de Melhoria no lobby principal e selecione a opção 'Transferência de Status'.
  </Step>
  <Step num="2" title="Selecione a Unidade de Origem">
    Escolha a unidade que possui atualmente os status de alto nível que você deseja mover.
  </Step>
  <Step num="3" title="Selecione a Unidade de Destino">
    Escolha a nova unidade, mais poderosa, que receberá os status. Observe que os status originais da unidade de destino serão substituídos.
  </Step>
  <Step num="4" title="Confirme a Transferência">
    Pague o custo necessário em ouro ou materiais para finalizar o processo. Sua nova unidade agora tem a distribuição de status otimizada.
  </Step>
</Steps>

| Tipo de Transferência | Nível de Custo | Resultado |
| :--- | :--- | :--- |
| **Normal para Normal** | Baixo | Os status são movidos, a unidade de origem permanece |
| **Shiny para Normal** | Médio | O destino torna-se Shiny, a origem é consumida |
| **Transferência Mítica** | Alto | Migração completa de status entre unidades de alto nível |

<Callout type="success" title="Nota de Estratégia">
Esta mecânica torna seguro investir em suas melhores unidades atuais. Você nunca "perde" progresso porque esses status sempre podem ser movidos para o próximo personagem definidor de meta que você conseguir.
</Callout>

</Section>

<Section icon={Shield} title="Traços e Otimização Crítica">

Além dos **status de Expedições de Anime** padrão, os Traços (também conhecidos como encantamentos) fornecem uma camada massiva de poder. Os Traços são separados do sistema de potencial e oferecem multiplicadores únicos que podem definir o papel de uma unidade em sua equipe. Por exemplo, uma unidade com alto alcance base, mas baixa velocidade, pode ser "consertada" com um traço orientado para velocidade.

O traço "Precisão" é atualmente um dos encantamentos mais procurados no meta de 2026. A Precisão II, por exemplo, oferece um aumento de 10% no Dano Crítico e um aumento de 20% na Chance Crítica. Quando combinados com status críticos base de nível S, suas unidades podem alcançar quase 100% de tempo de atividade crítico, dobrando efetivamente seu DPS em Raides.

| Nome do Traço | Bônus de Status | Raridade/Chance |
| :--- | :--- | :--- |
| **Precisão I** | +5% Dano Crítico, +10% Chance Crítica | Comum (15%) |
| **Precisão II** | +10% Dano Crítico, +20% Chance Crítica | Raro (4%) |
| **Rapidez** | +15% Velocidade de Ataque | Incomum (10%) |
| **Gigante** | +20% Alcance, +10% Dano | Raro (2%) |
| **Divino** | +50% Dano, +20% Velocidade | Secreto (&lt;1%) |

<Callout type="info" title="Índice de Traços">
Você pode visualizar a lista completa de encantamentos potenciais no Índice de Traços. Isso ajuda você a decidir se deve manter uma rolagem atual ou arriscá-la por um multiplicador melhor.
</Callout>

</Section>

<Section icon={Trophy} title="Progressão e Evolução de Unidade">

Para realmente maximizar seus **status de Expedições de Anime**, você deve buscar a Evolução e Ascensão. A Ascensão exige que você tenha várias cópias da mesma unidade Mítica. Ao sacrificar três duplicatas, você pode ascender sua unidade primária, o que concede um multiplicador massivo a todos os status base existentes.

| Estágio de Evolução | Requisitos | Multiplicador de Status |
| :--- | :--- | :--- |
| **Unidade Base** | Nenhum | 1.0x |
| **Ascendido I** | 3 Unidades Duplicadas | 1.5x |
| **Evoluído** | Materiais de Criação Específicos | 2.0x + Nova Habilidade |
| **Shiny Evoluído** | Status Shiny + Materiais | 2.5x + Efeitos Visuais |

O equipamento também desempenha um papel. Ao final das partidas, equipamentos podem cair aleatoriamente. Esses itens podem ser equipados para fornecer bônus fixos aos status da sua unidade, como +100 de Ataque ou +5% de Alcance. Assim como as unidades, os equipamentos podem receber reroll e ser bloqueados para garantir que você obtenha os melhores bônus possíveis para seu conjunto.
