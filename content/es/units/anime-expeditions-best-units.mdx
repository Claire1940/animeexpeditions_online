<Section icon={Shield} title="Pasos de evolución y progresión">

La forma base de una unidad rara vez es suficiente para afrontar el contenido del juego tardío. Para maximizar las **mejores unidades de Anime Expeditions**, debes participar en el sistema de evolución. La evolución no solo aumenta las estadísticas base, sino que a menudo añade mecánicas completamente nuevas, como el efecto de "Tambaleo" para Salmon Sorcerer o la "Forma Gigante" para Lady Giant.

<Steps>
  <Step num="1" title="Adquirir unidades base">
    Usa códigos activos y gemas para invocar unidades Míticas o Secretas. Enfócate en el banner actual que presenta a Shadow o 8th Sword.
  </Step>
  <Step num="2" title="Farmear esencias de evolución">
    Participa en incursiones y desafíos especializados para recolectar las esencias específicas requeridas para la evolución de tu unidad.
  </Step>
  <Step num="3" title="Equipar rasgos óptimos">
    Visita al PNJ de relanzamiento para buscar rasgos como **Godspeed** (Velocidad de ataque) o **Investor** (Dinero) según el rol de la unidad.
  </Step>
  <Step num="4" title="Obtener equipamiento característico">
    Completa misiones de eventos específicos para encontrar equipamiento que tenga sinergia con las pasivas de tu unidad, como el equipo True Love para Cursed Student.
  </Step>
</Steps>

| **Lady Giant Evo** | +55% de daño | Invoca muros de piedra (obstáculos) |
| **Elf Mage Evo** | +30% de velocidad | Ataques mágicos de seguimiento arcano |
