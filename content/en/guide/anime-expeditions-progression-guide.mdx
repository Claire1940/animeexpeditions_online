Effective resource management is a core pillar of the **anime expeditions progression guide**. The game utilizes a multi-banner system to ensure players can target specific units. | Banner Type | Refresh Rate | Pity Mechanic | Best For |
| :--- | :--- | :--- | :--- |

| **Beginner** | Event Based | 5 Multis (Guaranteed Mythic) | New Players |

If you are close to a pity threshold, it is often wise to wait for a Blitz event to maximize your return on investment.

<Section icon={Target} title="Evolution and Unit Trials">

Evolving your units is the most significant power jump in the **anime expeditions progression guide**. Each unit has a specific evolution path that unlocks new passives and abilities. Instead of traditional RNG-heavy grinding, Anime Expeditions introduces **Unit Trials**. Once you pull a unit, you can enter a preset trial. Beating this trial with the provided team rewards you with the necessary evolution item for that specific character.

<Steps>
  <Step num="1" title="Acquire Base Unit">
    Pull the unit from any active banner or earn them through events.
  </Step>
  <Step num="2" title="Access Unit Quest">
    Open the Quest tab and select the Unit Quest for your specific character.
  </Step>
  <Step num="3" title="Complete the Trial">
    Beat the challenge stage using the provided preset team to prove mastery.
  </Step>
  <Step num="4" title="Apply Evolution">
    Use the rewarded item in the Evolve menu to unlock the unit's full potential.
  </Step>
</Steps>

Evolution doesn't just increase stats; it often fundamentally changes how a unit functions. For example, the Salmon Sorcerer gains a support stun passive upon evolution, making him viable for late-game raids where crowd control is mandatory. Furthermore, evolved units gain a third equipment slot dedicated to their unique character-specific gear.

| Evolution Benefit | Description |
| :--- | :--- |
| **New Passives** | Unlocks unique abilities like stuns or burns |
| **Stat Multipliers** | Significant increase to base Damage and Range |
| **Equipment Slot** | Unlocks a 3rd slot for character-specific items |
| **Visual Upgrade** | Changes character model and effect colors |

<Callout type="success" title="Mastery Mode Drops">
